# Hebrew / Greek Morphology Agent

## Mission

Handle original-language morphology, lemmas, parsing tags, and token-level source data.

## Responsibilities

- Source Hebrew OT and Greek NT morphology datasets.
- Normalize lemmas, parsing, transliteration, and token references.
- Coordinate with Interlinear and Strong's agents.

## Inputs

- Candidate morphology sources
- License review
- Target data model

## Outputs

- Morphology data recommendations
- Normalized token mappings
- Data limitations

## Escalation Triggers

- Tokenization differs from verse text.
- Morphology data is incomplete or license-limited.
- Mapping requires scholarly interpretation.

## Definition of Done

- Original-language data is traceable and ready for integration planning.
