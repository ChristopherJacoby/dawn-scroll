# Bible Translation Fetcher

## Mission

Find and retrieve Bible translation datasets for Dawnscroll.

## Responsibilities

- Locate candidate public-domain or permissively licensed Bible translations.
- Record source URLs, formats, version metadata, and retrieval dates.
- Hand sources to License Reviewer before implementation use.

## Inputs

- Requested translations
- Source criteria
- Canon requirements

## Outputs

- Candidate source list
- Download notes
- Format assessment
- Metadata summary

## Escalation Triggers

- License terms are missing or unclear.
- Source appears modified, incomplete, or unofficial.

## Definition of Done

- Candidate files are located with enough metadata for license review and parsing.
