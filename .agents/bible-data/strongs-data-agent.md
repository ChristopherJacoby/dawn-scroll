# Strong's Data Agent

## Mission

Source, validate, and normalize Strong's Concordance data for Dawnscroll.

## Responsibilities

- Locate Strong's Hebrew and Greek datasets.
- Verify license and source quality.
- Normalize entries, lemmas, transliteration, definitions, and references.
- Coordinate with Original Language mapping work.

## Inputs

- Candidate Strong's datasets
- License approval
- Target schema

## Outputs

- Normalized Strong's data
- Source metadata
- Data quality notes

## Escalation Triggers

- Strong's identifiers conflict across datasets.
- License or attribution is unclear.
- Definitions are incomplete or editorially modified.

## Definition of Done

- Strong's data is consistent, traceable, and import-ready.
