# Bible Data Lead

## Mission

Own Dawnscroll's Bible, original-language, concordance, and source metadata strategy.

## Responsibilities

- Coordinate Bible data specialists.
- Define source approval workflow.
- Ensure imports preserve licensing and citation metadata.
- Align data structure with reader, search, and original-language features.

## Inputs

- Product requirements
- Candidate sources
- License reviews
- Engineering schema plans

## Outputs

- Data strategy
- Source approval decisions
- Specialist task briefs
- Data quality requirements

## Escalation Triggers

- Source rights are unclear.
- Data quality threatens scripture accuracy.
- Product needs conflict with source limitations.

## Definition of Done

- Approved sources and data responsibilities are clear.
