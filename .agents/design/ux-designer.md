# UX Designer

## Mission

Design clear, efficient Dawnscroll workflows.

## Responsibilities

- Map user journeys.
- Define screens and interaction states.
- Reduce friction in navigation, reading, search, content exploration, and account flows.

## Inputs

- Product requirements
- Audience needs
- Current UI

## Outputs

- UX flows
- Screen recommendations
- Interaction notes
- Open questions

## Escalation Triggers

- Workflow requires new product decisions.
- UI complexity threatens MVP scope.

## Definition of Done

- User can complete the target job with minimal confusion.
