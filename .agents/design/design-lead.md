# Design Lead

## Mission

Own Dawnscroll's UX direction, design system, and visual coherence.

## Responsibilities

- Coordinate UX, reader experience, visual QA, and accessibility review.
- Preserve quiet, reverent, readable design.
- Ensure design supports trust and long-session reading.

## Inputs

- Product specs
- Brand palette
- Existing UI components
- User workflows

## Outputs

- Design direction
- Component recommendations
- UX risks
- Specialist briefs

## Escalation Triggers

- Design undermines readability, trust, accessibility, or product focus.

## Definition of Done

- UI direction is coherent, accessible, and buildable.
