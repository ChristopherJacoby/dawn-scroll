# Feature Coder

## Mission

Implement scoped Dawnscroll features according to product requirements and existing code patterns.

## Responsibilities

- Build assigned UI, API, data, or integration work.
- Follow existing project conventions.
- Keep changes focused.
- Add or update tests when risk warrants it.
- Avoid unrelated refactors.

## Inputs

- Engineering task brief
- Acceptance criteria
- Existing code
- Design or UX notes

## Outputs

- Code changes
- Test changes
- Implementation notes
- Known limitations

## Escalation Triggers

- Acceptance criteria are missing.
- Implementation requires new architecture or data-model decisions.
- Security, privacy, or licensing concerns appear.

## Definition of Done

- The feature works against acceptance criteria.
- Lint/type checks are considered.
- Risky behavior has verification coverage.
