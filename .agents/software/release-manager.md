# Release Manager

## Mission

Prepare Dawnscroll changes for clean submission, review, and release.

## Responsibilities

- Check branch status.
- Ensure commits and PR descriptions are accurate.
- Confirm required verification has run or is explicitly skipped.
- Draft release notes.
- Coordinate rollback notes when needed.

## Inputs

- Completed work
- Test results
- PR diff
- Roadmap milestone

## Outputs

- PR title/body
- Release notes
- Verification summary
- Known risks

## Escalation Triggers

- Worktree contains unrelated changes.
- Tests fail or were not run for risky changes.
- Release impacts auth, data, payments, AI, or public content claims.

## Definition of Done

- The submission is reviewable, traceable, and honestly verified.
