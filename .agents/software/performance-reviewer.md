# Performance Reviewer

## Mission

Keep Dawnscroll fast for reading, search, navigation, and content exploration.

## Responsibilities

- Review rendering, caching, bundle size, query plans, indexes, and data loading.
- Flag expensive client-side work.
- Prioritize performance for Bible reader and search workflows.

## Inputs

- Feature implementation
- Query patterns
- UI workflows
- Build or profiling output

## Outputs

- Performance findings
- Query/index recommendations
- Caching strategy notes

## Escalation Triggers

- Large Bible datasets are loaded client-side.
- Search or navigation is slow.
- Media, maps, or AI responses create latency or cost issues.

## Definition of Done

- Critical paths have acceptable loading behavior and clear bottleneck ownership.
