# Bible Data & Scholarship Orchestra

## Mission

Source, license, normalize, validate, and cite Bible and original-language datasets.

## Default Flow

1. Bible Data Lead defines the source/data objective.
2. Translation Fetcher or Strong's/Hebrew/Greek agents locate candidate sources.
3. Translation License Reviewer approves or rejects usage.
4. Bible Parser / Canon Normalizer converts data into Dawnscroll format.
5. Data QA Agent validates completeness and structure.
6. Citation Integrity Reviewer checks source metadata.
7. Source Reliability Reviewer rates source quality when claims are involved.

## Use For

- Bible translations
- Strong's Concordance
- Hebrew/Greek morphology
- Interlinear data
- Verse normalization
- Citation metadata
- Source reliability checks

## Required Outputs

- Approved source list
- License status
- Normalization notes
- Data QA report
- Citation metadata requirements
