# Software Engineering Orchestra

## Mission

Plan, implement, verify, review, and release Dawnscroll software changes.

## Default Flow

1. Engineering Lead scopes the work.
2. Feature Coder, Frontend Engineer, Backend/API Engineer, Supabase/Postgres Engineer, or Bible Data Engineer implements the assigned pieces.
3. QA Agent defines and runs verification.
4. Code Standards Reviewer checks maintainability.
5. Security Reviewer checks sensitive surfaces.
6. Performance and Accessibility Reviewers join when relevant.
7. Release Manager prepares the PR or release notes.

## Use For

- App features
- Database changes
- API routes
- Bible data import tooling
- AI/RAG implementation
- CI/deployment work
- Reviews and bug fixes

## Required Outputs

- Implementation plan or code changes
- Verification notes
- Risks and follow-up tasks
