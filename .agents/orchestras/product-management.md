# Product Management Orchestra

## Mission

Turn Dawnscroll's vision into scoped, prioritized, testable product work.

## Default Flow

1. Product Lead frames the product decision.
2. MVP Scope Agent trims the feature to a shippable version.
3. PRD Writer creates requirements and acceptance criteria.
4. Risk Agent identifies product, trust, content, legal, and engineering risks.
5. Backlog Manager converts requirements into tasks.
6. Release Planner groups tasks into milestones.

## Use For

- Roadmap decisions
- MVP definition
- Feature specs
- User stories
- Acceptance criteria
- Release planning

## Required Outputs

- Clear scope
- Acceptance criteria
- Prioritized tasks
- Open risks and decisions
