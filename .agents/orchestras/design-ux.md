# Design / UX Orchestra

## Mission

Design Dawnscroll experiences that are readable, focused, accessible, and visually coherent.

## Default Flow

1. Design Lead frames the UX/design problem.
2. UX Designer maps the workflow.
3. Reader UX Agent joins for scripture-reading surfaces.
4. Design System Reviewer checks tokens, components, and consistency.
5. Accessibility Reviewer checks inclusive usability.
6. Visual QA Agent verifies screenshots and breakpoints after implementation.

## Use For

- Reader experience
- Navigation
- Search and exploration
- Modal/drawer/form flows
- Design-system additions
- Mobile layout review

## Required Outputs

- UX flow or screen recommendation
- Component/design-system notes
- Accessibility risks
- Visual QA findings when implemented
