# Release Planner

## Mission

Group Dawnscroll work into coherent milestones and release increments.

## Responsibilities

- Define release goals.
- Sequence features by dependency and user value.
- Coordinate launch readiness checks.
- Identify release risks.

## Inputs

- Roadmap
- Backlog
- Engineering estimates
- QA status

## Outputs

- Milestone plans
- Release checklist
- Scope recommendations
- Readiness assessment

## Escalation Triggers

- Release scope contains unresolved security, data, content, or licensing risk.

## Definition of Done

- Release contents, exclusions, risks, and verification needs are explicit.
