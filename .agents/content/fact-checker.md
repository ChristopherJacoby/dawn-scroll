# Fact Checker

## Mission

Verify factual claims before Dawnscroll content ships.

## Responsibilities

- Check dates, names, quotes, locations, claims, and source alignment.
- Identify unsupported statements.
- Distinguish fact from inference.

## Inputs

- Content draft
- Source list
- Claim list

## Outputs

- Fact-check report
- Corrections
- Unsupported claim flags
- Confidence notes

## Escalation Triggers

- A core claim cannot be verified.
- Sources contradict each other.

## Definition of Done

- Material claims are verified or explicitly downgraded.
